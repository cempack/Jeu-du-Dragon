À PROPOS DU JEU

Le Jeu Du Dragon est un jeu vidéo en 2D développé en Python avec Pygame, créé pour le concours Trophées NSI 2025 par deux lycéens français au lycée Edmond Perrier de Tulle.

Explorez divers niveaux, incarnez différents héros avec des capacités uniques, et plongez dans un monde de pixels où dragons et aventures vous attendent !

TECHNOLOGIES UTILISÉES

  • Python : Langage de programmation.
  • Pygame : Bibliothèque pour le gameplay.
  • GitHub : Gestion du code source.

INSTALLATION

1. Clonez le projet :
   git clone https://github.com/cempack/Jeu-du-Dragon.git
   cd Jeu-du-Dragon
   cd jeu
   cd sources

2. Installez les dépendances :
   pip install -r requirements.txt

3. Lancez le jeu :
   python main.py
